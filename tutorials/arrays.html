Arrays

A set of variables

Computer programming can always be broken down into the processing of variables, but often the number of variables you need becomes impractical to declare and assign each one.

Take a classroom of 30 students. If you were to make a String variable for each student, you’d already have 30 lines of code, one for each variable with a distinct name from the others. As well, searching for a name would take at least lines of code to implement, and sorting would be impossible. As well, especially in competitive programming, you are given the number of variables you have to store; you cannot hardcode it. Thus, a solution was given: the array.

An array is a collection of items, all grouped together under one name. The items themselves, instead of having identifiers, are indexed from 0 onwards.

Implementation

Let’s show this with an example: create an attendance list for the names of a class of n people, where n is an integer greater than 0.

String[] attendance = new String[n];

Here we initialize our array. Because arrays are not primitive variables, they need the keyword new. As well, the n is put in the square brackets to make sure the length of the array (the number of items) is equal to n. Of course, if you don’t want a String and want some other variable type, just replace the String with the name of that type (i.e. int[] arr = new int[n] for integer).

Congratulations! You just created an empty array!

You can imagine an array as a line of empty slots, numbered from 0 to n-1 (note that the length of the array is still n-1-0+1=n). Now let’s fill them. The first student on our list is Aaron, so let’s put that in.

attendance[0] = “Aaron”;

Each slot in the array behaves like a variable, and thus the syntax is about the same. The main difference is that, in place of an identifier or name, use the name of the array followed by the index within square brackets.

Having said that, reading from an array from an index is the same as reading the variable value:

String read = attendance[0];

Other Methods

  • .length: returns the length of the array (e.g. array.length).

  • Arrays.sort(): returns a sorted version of the array using Quicksort (e.g. Arrays.sort(array)).
