2D Arrays

Transcending dimensions with arrays

Arrays make it possible to group and index multiple objects together under one name. But what if the thing you want to group is an array?

Implementation

int[] [] arr = new int[rows] [columns];

We have established that 1D arrays can be represented as a line of “slots”, each slot representing a variable.

Then what if each slot contained an array?

int[] [] arr = new int[4] [3];

Now we have a 4 by 3 array, which we could interpret as an array consisting of 4 arrays, each with length 3.

Then let’s put a value inside.

arr[0] [2] = 69;


Similar to 1D arrays, accessing the individual values in a 2D array only requires the array name as well as the indices of the value. Remember that the first index is for row, and the second for column.

Other Methods

In 1D arrays, using .length returns the length of the array (the number of elements). In 2D arrays, that same method only returns the number of rows in the array (remember that a 2D array is just an array of arrays). To find the number of elements in a row n, you can use:

arr[n].length

Having said that, it is possible for every row in an array to have different lengths.
